# ふわふわシフォンケーキ - シフォンケーキ専門店ウェブサイト

## プロジェクト概要
シフォンケーキ専門店のための、シンプルで美しい1ページ完結型のウェブサイトです。HTML、CSS、JavaScriptを1つのファイルにまとめたシンプルな構造になっています。

## 完成した機能

### ✅ 実装済み機能
1. **レスポンシブデザイン**
   - スマートフォン、タブレット、デスクトップに対応
   - 画面サイズに応じて最適なレイアウトに自動調整

2. **ナビゲーションメニュー**
   - 固定ヘッダーで常にアクセス可能
   - スムーススクロール機能搭載
   - セクションへのリンク（お店について、メニュー、こだわり、アクセス）

3. **メニューセクション**
   - 6種類のシフォンケーキを紹介
   - プレーン、チョコレート、抹茶、いちご、キャラメルナッツ、オレンジ
   - 各商品に価格とアイコン、説明を表示
   - ホバー時のアニメーション効果

4. **こだわりセクション**
   - 4つの特徴をカードで表示
   - 新鮮な卵、厳選素材、手作りの温もり、ギフト対応

5. **アクセス情報**
   - 店舗名、住所、電話番号、営業時間、定休日などを明示
   - 見やすい定義リスト形式で表示

6. **インタラクティブ要素**
   - スクロールアニメーション（フェードイン効果）
   - ホバーエフェクト（カードが浮き上がる）
   - スムーススクロール

7. **視覚的デザイン**
   - 温かみのあるベージュ系のカラーパレット
   - グラデーション効果
   - 絵文字アイコンでわかりやすく親しみやすい印象

## ファイル構成

```
index.html          # すべてのHTML、CSS、JavaScriptを含む単一ファイル
README.md           # プロジェクトドキュメント
```

## 機能エントリーポイント

### メインページ
- **パス**: `index.html`
- **セクション**:
  - `#about` - お店について
  - `#menu` - メニュー
  - `#features` - こだわり
  - `#access` - アクセス情報

## 使用技術

- **HTML5**: セマンティックなマークアップ
- **CSS3**: 
  - Flexbox、Grid Layout
  - アニメーション、トランジション
  - グラデーション
  - メディアクエリ（レスポンシブデザイン）
- **JavaScript (Vanilla JS)**:
  - Intersection Observer API（スクロールアニメーション）
  - イベントリスナー（スムーススクロール、ホバー効果）

## カスタマイズポイント

### 店舗情報の変更
`index.html` の以下のセクションを編集してください：
- ヘッダー部分: 店舗名
- アクセスセクション: 住所、電話番号、営業時間など

### メニューの追加・変更
`.menu-item` の要素を追加・編集することで、商品を変更できます。

### カラースキームの変更
CSS変数を使用していないため、以下の色コードを検索・置換してください：
- `#8b6f47` - メインのブラウン色
- `#d4a574` - アクセントカラー
- `#f8e8d8` - 背景グラデーションの開始色

## 推奨される次のステップ

1. **実際の商品写真の追加**
   - 高品質な商品写真を撮影し、メニューカードに追加
   - ヒーローセクションに魅力的なメイン画像を配置

2. **お問い合わせフォームの実装**
   - 顧客からの問い合わせを受け付けるフォーム
   - FormspreeやGoogle Formsとの連携

3. **SNS連携**
   - Instagram、Facebook、TwitterなどのSNSボタン追加
   - Instagram APIで商品写真の自動表示

4. **Google Maps埋め込み**
   - アクセスセクションに地図を追加

5. **オンライン注文機能**
   - RESTful Table APIを使用した予約管理システム
   - カート機能の実装

6. **季節限定メニューの表示**
   - 期間限定商品を目立たせる仕組み

## デプロイ方法

ウェブサイトを公開する場合は、**Publishタブ**から1クリックでデプロイできます。
